# Doublependulum
Tarkoituksena mallintaa kahta toisissaan kiinni olevaa heiluria. 
Kaksoisheiluri on esimerkki ns. kaaottisesta systeemistä, jossa pienet
muutokset alkuasetelmissa luovat arvaamattomia seurauksia.
